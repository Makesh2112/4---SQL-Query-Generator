# 4---SQL-Query-GeneratorProject Description
SQL Query Generator is an intelligent database query assistant that leverages IBM's Granite 3.3 2B Instruct model to convert natural language requests into safe, production-ready SQLite SQL queries. The platform addresses the challenge of SQL complexity by delivering AI-powered query generation, intelligent table and column detection, safety validation, and comprehensive export capabilities including PDF reports, text files, and social media sharing.Core Technologies
•   	Python 3.x: Core programming language for application logic
•   	IBM Granite 3.3 2B Instruct: AI model for natural language understanding
•   	Gradio: Interactive chat interface framework
•   	SQLite: Lightweight relational database for query generation
•   	ReportLab: PDF generation library for professional reports
•   	Google Colab: Cloud-based development and deployment platform
•   	Regular Expressions: Pattern matching for intelligent parsing
